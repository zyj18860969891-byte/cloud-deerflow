## 💻 常用开发命令

```bash
# ========== 环境管理 ==========
make check              # 检查环境
make install            # 安装依赖
make dev                # 启动服务 (推荐首先运行)
make stop               # 停止服务

# ========== 后端开发 ==========
cd backend
make test               # 运行测试
make lint               # 代码检查
ruff check . --fix      # 自动修复
uv add package_name     # 添加依赖

# ========== 前端开发 ==========
cd frontend
pnpm dev                # 开发服务
pnpm test               # 运行测试
pnpm lint --fix         # 自动修复
pnpm build              # 构建生产版

# ========== Docker 开发 ==========
make docker-dev         # Docker 启动
docker-compose logs -f  # 查看日志

# ========== 日志查看 ==========
tail -f logs/*.log      # 实时日志
```
